# 创建 API Token

## 什么是 Token

Token 是调用 SooAPI 的凭证，类似于密码。请妥善保管，不要泄露给他人。

## 创建 Token

1. 登录 SooAPI 控制台
2. 进入「API Token」页面
3. 点击「创建 Token」
4. 设置 Token 名称（建议填写用途，如 `claude-code-work`）
5. 复制生成的 Token

::: warning 注意
Token 只显示一次，创建后请立即保存！
:::
