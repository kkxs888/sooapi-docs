# 常见问题

## 账户问题

### Token 无法登录

- 确认 Token 拼写正确
- 检查 Token 是否已过期
- 尝试重新创建 Token

### 余额为负

余额为负表示已透支，请及时充值。

## 调用问题

### 报错 "No available channel"

可能原因：
1. 该模型未在渠道中配置
2. 渠道额度已用完
3. 渠道被禁用

解决方法：检查渠道配置或联系客服。

### 报错 "Invalid token"

- 检查 API Key 是否正确
- 确认 Authorization 头格式：`Bearer your-token`

### 响应时间很长

可能原因：
1. 网络问题
2. 模型队列繁忙
3. 请求内容过大

## 费用问题

### 如何查看账单

在控制台的「用量统计」中可以查看详细账单。

### 如何节省费用

1. 优先使用轻量模型（如 `gpt-4o-mini`）
2. 开启缓存减少重复请求
3. 设置用量告警
